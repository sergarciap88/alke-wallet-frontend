# 👛 Alke Wallet - Frontend

¡Bienvenido al repositorio de **Alke Wallet**! Este proyecto es una aplicación web de billetera digital desarrollada como parte de la evaluación del **Módulo 2: Fundamentos del desarrollo Front-End** de Alkemy.

---

## 🚀 Descripción del Proyecto

La aplicación permite a los usuarios gestionar sus activos financieros de manera sencilla e intuitiva a través de una interfaz dynamic y responsive. 

### ⚙️ Funcionalidades Principales
* **Inicio de Sesión (`login.html`):** Autenticación simulada de credenciales.
* **Menú Principal (`menu.html`):** Visualización del saldo disponible y accesos rápidos a las operaciones.
* **Depósitos (`deposit.html`):** Formulario para simular recargas de dinero a la cuenta y actualización de saldo en tiempo real.
* **Envío de Dinero (`sendmoney.html`):** Transferencias a contactos registrados con validación de saldo disponible.
* **Historial de Movimientos (`transactions.html`):** Tabla interactiva con el registro completo de ingresos y egresos.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica de las vistas.
* **CSS3:** Estilos personalizados adicionales.
* **Bootstrap 5:** Framework CSS para diseño responsivo y componentes visuales.
* **JavaScript (ES6):** Lógica del negocio y gestión de `localStorage`.
* **jQuery:** Manipulación eficiente del DOM y manejo de eventos.
